Mirage 0.8.3 Released

Another minor bugfix release. Unfortunately progress has stagnated as
I've been pretty busy between my thesis work and another application.
There are still a number of features I want to implement for the 0.9
release, and hopefully I will be able to begin work on them in another
month or so.

CHANGELOG

- Use F11 for fullscreen like most applications
- Use 'kfmclient' to launch browser in kde
- Added Italian translation (Daniele Maggio)
- Bug: Save As doesn't work
